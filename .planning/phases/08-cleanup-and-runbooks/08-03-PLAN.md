| Boundary | Description |
|----------|-------------|
| Human operator <-> `TFX_PAT` (revoke / restore) | The SC #5 exercise deliberately revokes the Marketplace PAT and mints a fresh one â€” the fresh PAT must be created with minimal scope and pasted into the secret, never committed, never echoed; the old token must be revoked, not left dangling |

| GitHub App private key <-> the workflow | Now live for the first time â€” the key sits in `APP_PRIVATE_KEY`; if it leaks, an attacker can push to `release` bypassing its (light) rules â€” least-privilege App install + 1-hour auto-revoked token + repo-scoped contain the blast radius |
