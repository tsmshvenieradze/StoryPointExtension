- **E2E flow:** all 13 documented steps (feature PR → master → `master→release` promotion PR → `push:[release]` → mint App token → gates → in-memory bump → tfx create → upload-artifact → tfx publish → App commit-back to `release` → annotated tag → open `release→master` back-merge PR → operator squash-merges) are implemented in `publish.yml`.
